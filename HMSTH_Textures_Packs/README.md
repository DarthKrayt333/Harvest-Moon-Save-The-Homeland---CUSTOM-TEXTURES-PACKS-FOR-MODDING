# HMSTH Texture Source Repository

This repository contains **custom changed .bmp texture files** for **Harvest Moon: Save the Homeland** (PS2).

These textures are intended to be used with **[HMSTHModdingTool](https://github.com/DarthKrayt333/HMSTHModdingTool)**.

### NPC Textures
- Changed textures for various villagers

### Other Textures
- Some changed interior house textures
- Some of animal textures

Additional texture packs will be added in the future.

---

## How to Use

These are **source .bmp files**, not pre-built `.GDTB` archives.

**Installation Steps:**

1. Download the desired texture pack.
2. Extract the `.bmp` files.
3. Use **HMSTHModdingTool** to extract the original `.HDA` file (example: `BOY.HDA` or `COMMON.HDA`).
4. Extract the target `.GDTB` file using the `-xgdtb` command.
5. Replace the original `.bmp` files with the changed ones from this repository.
6. Rebuild the `.GDTB` file using the tool.
7. Repack the `.HDA` and replace it with your game’s `.HDA` file.
